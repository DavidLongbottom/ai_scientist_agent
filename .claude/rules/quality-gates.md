# Quality Gates

**审稿分数 ≥ 5/10 可提交，< 3/10 必须重做**

## LLM Review Thresholds

| Score | Gate | Action |
|---|---|---|
| >= 5/10 | Accept | Proceed to final compilation and reporting. |
| 3.0 to 4.9/10 | Borderline | Consider adding quick fixes or running an extra metric if cheap. |
| < 3/10 | Reject | The idea or execution failed significantly. Re-roll idea or perform an extra debug loop. |

No paper can be marked "completed" successfully if it scores <3.
