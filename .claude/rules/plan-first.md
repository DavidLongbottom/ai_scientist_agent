# Plan-First Workflow

**实验前必须有假设和设计**

## The Protocol

1. **Enter Plan Mode** — Stop writing code immediately.
2. **Check MEMORY.md** — Find any relevant `[LEARN]` entries for this idea.
3. **Write the Hypothesis** — Formulate a clear "if... then..." statement for the experiment.
4. **Draft the Design** — Which metrics to track, which baseline to compare against.
5. **Save to disk** — `experiments/plans/YYYY-MM-DD_short_idea_name.md`.
6. **Execute** — After plan is finalized, execute the `/tree-search` or implementation.
